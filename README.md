# Aircraft Performance & Mission Simulator

This is a research-oriented aircraft performance simulation tool built using Python and Streamlit.

## Features

- ISA atmosphere model
- Thrust lapse modeling
- Climb performance
- Service ceiling estimation
- Time-to-climb calculation
- Breguet range estimation
- Engine thrust vs Mach plot

## Run Locally

1. Install dependencies:

pip install -r requirements.txt

2. Run the app:

streamlit run main.py

## Author

Developed for research and academic purposes in aircraft performance analysis.
